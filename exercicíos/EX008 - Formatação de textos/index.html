Principais formatações

Negrito / Destaque

Existem duas formas de colocar negrito, sendo não semântica ou com semântica

Nesta frase temos um termo em negrito usando a tag <b> (não semântica)
Nesta outra frase temos um um outro termo em negrito com a tag <strong> (semântica)

Itálico / Enfâse

Nesta frase temos um termo em itálico com a tag <i> (não semântica)
Nesta outra, temos um outro termo em itálico com a tag <em> (semântica.)

Sublinhado

Podemos marcar um texto com Sublinhado usando a tag <u> (não semântica)

Marcado

E aqui, nós vemos um termo marcado com a tag <mark>

E também temos um outro texto marcado, feito com <mark> mas com uma cor que configurei para estilo geral!

colocando a tag <style>

Texto grande e pequeno

Temos aqui um texto grande e um texto pequeno feitos com a tag < e > sendo ambas formas não semânticas

Texto deletado

Podemos marcar um texto como deletado, ou excluído pra dizer que deve ser lido, mas não considerado. Usando a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar enfâse e dizer que foi inserido somente depois. Usando a tag < >

Sobrescrito e subescrito

O sobrescrito é usado para coisas como x20+3 E subescrito como log2X
